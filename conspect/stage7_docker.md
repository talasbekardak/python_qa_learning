  Docker — платформа для запуска приложений в изолированных контейнерах. Контейнер работает одинаково на любой ОС. 
  Dockerfile - пишет инструкцию длч создания образа 
  docker build - создает образ 
  docker run - создает и запускает контейнер из образа 
  Docker для qa главная идея: тесты работают одинаково на любом компьютере, не нужно вручную устанавливать
  Python, библиотеки, браузер. Убирает проблему "у меня работает, у тебя нет".
